These notebooks provide the steps of implementing a predictive maintenance model found in the Predictive Maintenance Modeling Guide collection in the Cortana Intelligence Gallery. To execute the notebook code, you will need to open the Predictive Maintenance Modeling Guide Data Sets experiment in Azure ML Studio and save the datasets contained in that experiment to your workspace. Detailed instructions on how to save and import the datasets are provided below.

These are adaptations of a notebook originally created by Fidan Boylu Uz (a Microsoft employee) for distribution on the Cortana Intelligence Gallery.
